.NET Framework 官方離線版，獨立安裝包(4.7~1.0全版本)

 1 個月 前

.NET Framework 是 Microsoft 所提供既完整又一致的程式撰寫模型，可用來建置提供絕佳視覺效果使用者經驗、順暢安全通訊，而且能夠建立各種商務流程模型的應用程式。

Windows內建版本

  • Windows 10   自帶  .NET Framework 4.6
  • Windows  8.1 自帶  .NET Framework 4.5.1
  • Windows 8     自帶  .NET Framework 4.5
  • Windows 7     自帶  .NET Framework 3.5(2.0/1.1)

若你不清楚你已經安裝的 .NET Framework 有哪些版本可以透過 .NET Version Detector 這個軟體來查看

除非你有特殊用途，不然一般使用者推薦參考以下版本安裝即可

Windows 10 、Windows 8.1、Windows 8 只需安裝  .NET Framework 3.5

Windows 7 安裝 .NET Framework 4.X最後版本

也可以參考微軟官方說明https://msdn.microsoft.com/zh-tw/library/hh925568(v=vs.110).aspx
